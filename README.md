# jf: Simple JSON Formatter #

## Introduction ##

- Overwrite ugly JSON file(s) with prettier JSON file(s)

## Warning ##

- Do not use it on very large json file!

## Build ##

```
make
```

## Install ##

```
sudo make install
# or move bin/jf to your PATH
```

## Usage ##

```
jf a.json b.json ...
```
